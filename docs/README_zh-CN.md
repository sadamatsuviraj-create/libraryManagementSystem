## 项目简介

**libraryManagementSystem** 是一个基于 SQLite 数据库的本地终端图书管理系统，同时提供Web页面可视化管理。系统采用 C++20 标准开发，使用 CMake 构建，支持 Windows、Linux 和 macOS 三大主流操作系统。

系统通过管理员账号和普通用户（学生）账号实现分级权限管理：

| 角色 | 功能 |
|------|------|
| **管理员** | 图书管理（增删改查）、用户管理、借阅记录查询 |
| **普通用户（学生）** | 图书查询、借阅/归还/续借、个人账户管理 |

## 主要特性

| 特性 | 说明 |
|------|------|
| 跨平台支持 | Windows、Linux、macOS 原生运行 |
| 零外部依赖 | 内置 SQLite，无需额外安装数据库 |
| 密码安全 | SHA-256 加密存储用户密码 |
| 安全口令 | 支持设置找回密码的安全令牌 |

| **Web 界面** | 现代化的 Vue 3 WebUI |
| **RESTful API** | Node.js Express API 服务器用于 Web 集成 |
| **日志系统** | 全面的日志记录和查询功能 |
| **增强管理面板** | 全面的图书、用户和借阅管理功能 |
| **用户管理** | 完整的用户注册、认证和档案管理 |
| **响应式设计** | 为桌面和移动设备优化 |

## 项目构建

### 前置要求

| 依赖 | 版本要求 |
|------|----------|
| CMake | >= 3.10 |
| C++ 编译器 | 支持 C++20 标准 |

编译器选择：
- **Windows**: MinGW-w64 或 Visual Studio 2019+
- **Linux**: g++ >= 9.0
- **macOS**: Apple Clang 或 gcc

检查 CMake 是否已安装：
```bash
cmake --version
```

---

### 构建方式一：CMake 命令行

#### Linux / macOS

```bash
# 进入项目目录
cd /path/to/libraryManagementSystem

# 创建构建目录并编译
cmake -B build && cmake --build build

# 运行程序
cd build
./lib
```

#### Windows (MinGW-w64)

```bash
# 进入项目目录
cd /d /path/to/libraryManagementSystem

# 创建构建目录并编译
cmake -B build -G "MinGW Makefiles" && cmake --build build

# 运行程序
cd build
lib.exe
```

#### Windows (Visual Studio)

```bash
# 创建 Visual Studio 解决方案
cmake -B build -G "Visual Studio 17 2022"

# 编译（或使用 IDE 打开 build 目录）
cmake --build build --config Release

# 运行程序
build\Release\lib.exe
```

---

### 构建方式二：IDE

| IDE | 步骤 |
|-----|------|
| **CLion** | 打开项目 → 自动识别 CMake → 点击运行 |
| **Visual Studio** | 打开文件夹 → 选择启动项 `lib.exe` → 运行 |

> **注意**：如使用 VC++ 6.0 等老旧 IDE，建议改用 CMake 或现代 IDE

---

### 构建方式三：交叉编译

在 macOS/Linux 上编译 Windows 可执行文件：

#### macOS

```bash
# 安装依赖
brew install cmake mingw-w64

# 创建构建目录
mkdir build && cd build

# 配置交叉编译
cmake .. -DCMAKE_TOOLCHAIN_FILE=../cmake/Windows-x86.cmake

# 编译
cmake --build .
```

#### Linux (Ubuntu/Debian)

```bash
# 安装依赖
sudo apt-get update
sudo apt-get install -y cmake g++-mingw-w64-x86-64 gcc-mingw-w64-x86-64

# 创建构建目录
mkdir build && cd build

# 配置交叉编译
cmake .. -DCMAKE_TOOLCHAIN_FILE=../cmake/Windows-x86.cmake

# 编译
cmake --build .
```

### 核心功能说明

#### 1. 图书管理（管理员）

| 功能 | 说明 |
|------|------|
| 录入图书 | 添加新书，包含 ISBN、书名、作者、出版社、分类、数量 |
| 修改图书 | 根据 ISBN 修改图书信息 |
| 删除图书 | 删除指定图书（需无借阅记录） |
| 查找图书 | 按关键词（书名/作者/ISBN）搜索 |
| 列出所有 | 显示全部图书，支持按书名/作者/ISBN 排序 |

#### 2. 借阅管理（学生）

| 功能 | 说明 |
|------|------|
| 借阅图书 | 输入 ISBN 和借阅天数（1-90 天） |
| 归还图书 | 根据记录 ID 归还已借图书 |
| 续借图书 | 延长借阅期限 |
| 查看借阅 | 查看个人当前借阅记录 |

#### 3. 账户安全

| 功能 | 说明 |
|------|------|
| 修改密码 | 管理员可修改任意用户密码；学生只能修改自己密码 |
| 安全口令 | 设置找回密码的令牌，用于忘记密码时验证身份 |
| 忘记密码 | 通过用户名和安全口令重置密码 |

#### 4. 逾期管理

- 系统自动检测逾期图书
- 学生登录时如有逾期图书会主动提示
- 管理员可查看所有借阅记录及逾期状态

#### 5. 语言切换

- 支持中英文界面切换
- 语言偏好保存在 config.ini 文件中
- 从主菜单（选项 4）进入语言选择

## 使用说明

### 首次运行

程序首次运行时会自动：

1. 创建 `library.db` 数据库文件
2. 初始化表结构（图书表、用户表、借阅记录表）
3. 创建默认管理员账户（用户名：`admin`，密码：`admin`）

### 默认账户

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | admin |

### 学生注册流程

1. 在主菜单选择 `2. 学生注册`
2. 输入学号、姓名、学院、班级等信息
3. 设置登录用户名和密码
4. 注册完成后可登录系统

### 密码找回流程

1. 在主菜单选择 `3. 忘记密码`
2. 输入用户名
3. 输入之前设置的安全口令
4. 设置新密码

### 语言切换

1. 在主菜单选择 `4. 选择语言 / Choose Language`
2. 选择 `1. English` 或 `2. 中文`
3. 界面语言立即改变
4. 语言偏好保存在 config.ini 文件中

## 注意事项

| 项目 | 说明 |
|------|------|
| **数据库文件** | `library.db` 保存在运行目录，请勿删除 |
| **配置文件** | `config.ini` 存储语言偏好和其他设置 |
| **终端编码** | 建议使用 UTF-8 编码（Windows: `chcp 65001`） |
| **借阅限制** | 借阅天数 1-90 天，图书数量不能为负 |
| **删除限制** | 有借阅记录的图书无法删除 |
